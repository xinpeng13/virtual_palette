Installation

Download the file virtual_palette.js from our github repository

Include the following codes in the <head> section. Since our library utilized jquery, make sure to include the second line below.

<script defer type="text/javascript" src="virtual_palette.js"></script>

<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

Usage 1: Painting Palette

End users can:

1. View the hexadecimal code of the paint in the painting set.

2. Add color to the mix by clicking the painting set buttons on the left bar.

3. Delete color by clicking the color entries on the top of the palette.

4. Delete all color entries on the top of the palette by clicking the clear button.

5. Add the mixed color to the painting set by clicking on the mixed color.

6. View the info of color mixing by clicking the info button on the top-right corner.

(Additional functionalities will be explained in the following examples)

Basic Painting Palette Example

This is a basic non-popup painting palette example. It has default color theme and default usage, "painting". The paintset added is a pre-defined set named "12color".

<div id="example1">

</div>

// create a new PaletteGenerator that is not in a popup format

// the first boolean argument inside PaletteGenerator indicates whether you want to create a popup palette

const pg_12 = new PaletteGenerator(false)

// add the pre-defined 12 colors paintset '12color' to the PaletteGenerator

pg_12.addPaintSet("12color")

// add customized title "12 COLOR PAINTSET" to the PaletteGenerator

pg_12.addTitle("12 COLOR PAINTSET")

// find the place on the html that you want to insert the palette

const ptr1 = document.getElementById("example1")

// append this.show to the place you want to insert the palette

ptr1.appendChild(pg_12.show)

Pop Up Customized Painting Palette Example

This is a pop up customized painting palette example. It has "dark" color theme and default usage, "painting". Developer can customize the title of the painting set and the color that added.

<div id="example2">

</div>

// create a new popup PaletteGenerator

const pg_purple = new PaletteGenerator(true)

// add your own customized colors (in hexadecimal format) to the PaletteGenerator

pg_purple.addColor("#ffccff")

pg_purple.addColor("#ccccff")

pg_purple.addColor("#99ccff")

pg_purple.addColor("#66ccff")

pg_purple.addColor("#33ccff")

// add customized title to the PaletteGenerator

pg_purple.addTitle("Customized PAINTSET"))

// select theme "dark"

pg_purple.selectTheme("dark")

const ptr2 = document.getElementById("example2")

ptr2.appendChild(pg_purple.show)

Usage 2: Watercolor Palette

End users can:

1. Have all the functionalities in Usage 1

2. Alter the water level to the mixed color

3. Select the background color where the paint will be add on by clicking on the button on the bottom-right corner
